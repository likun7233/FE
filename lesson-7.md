# 第7课 Javascript基础

## 课程目标
认识javscript，掌握javascript的引入方式，熟悉基本数据类型，常量变量，掌握基本运算符的使用

## 课程内容

##### 1.认识Javascript
- javascript !== java
- 是客户端脚本语言

##### 2.Javascript引入方式
- \<script>开始,遇到\</script>结束
- \<script type="text/javascript" src="1.js">\</script>
- 标签的位置，放在body内的最下位置。区别css位置
- defer/async (了解)

4. JavaScript的基本运算符
